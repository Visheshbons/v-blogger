# Contributing to V-Blogger

Thank you for considering contributing to V-Blogger!

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Open an issue** to discuss your proposed change before making it.
3. **Write clear, concise commit messages**.
4. **Test your code** before submitting.
5. **Open a pull request** and describe your changes.

## Coding Guidelines

- Use clear, readable code and comments.
- Follow the existing code style.
- Keep pull requests focused and minimal.

## Reporting Issues

- Search for existing issues before opening a new one.
- Provide as much detail as possible (steps to reproduce, environment, etc.).

## Code of Conduct

Be respectful and constructive in all interactions.

---

Thank you for helping improve V-Blogger!
